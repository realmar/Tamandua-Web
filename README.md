Tamandua Web
============

Getting Started
---------------

### Prerequisites

 - [NodeJS](https://nodejs.org/en/)
 - [NPM](https://www.npmjs.com/)

### Installing

```sh
$ npm install

# serve on local machine
$ npm start
```

### Building

```sh
$ npm run build-prod        # create a production build
$ npm run build-stage       # create a stage build
```
